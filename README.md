# Django Course Management System (CMS)

This project is a **Course Management System (CMS)** built with Django 4. It allows instructors to create and manage courses and their content efficiently.

## Features

The CMS provides the following functionality:  

- **Course Creation and Management**  
  Instructors can create courses, add modules, and manage the content of each module.  

- **Class-Based Views and Mixins**  
  Uses Django's class-based views and mixins for clean and reusable code.  

- **Forms and Model Formsets**  
  Custom forms and model formsets allow editing modules and module content directly from the interface.  

- **Groups and Permissions**  
  User groups and permissions control access to different parts of the system.  

- **Drag-and-Drop Functionality**  
  Modules and module contents can be reordered using drag-and-drop for better content organization.  

- **User Authentication**  
  Supports user registration, login, and authentication for instructors and students.
